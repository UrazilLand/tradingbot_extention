# 크롬 익스텐션 기반 암호화폐 자동매매 봇

## 프로젝트 개요
사용자가 버튼 위치를 지정하고 매크로를 구성할 수 있는 크롬 익스텐션 기반 암호화폐 자동매매 시스템입니다. 볼린저 밴드 전략을 기반으로 하며, 다양한 거래소에서 작동합니다.

## 주요 특징
- **매크로 기반**: 사용자가 직접 버튼 위치 지정 및 매크로 구성
- **다양한 매매 전략**: 볼린저 밴드, RSI, MACD, 이동평균선, 스토캐스틱 등
- **다중 거래소 지원**: Gate.io, Binance, Upbit, Coinbase 등
- **실시간 알림**: 브라우저 알림 및 팝업 알림
- **사용자 친화적**: 직관적인 UI로 쉬운 설정
- **데이터 저장**: 로컬 스토리지 기반 설정 및 거래 내역 저장

## 핵심 전략 상세
### 볼린저 밴드 설정
- **기간**: 20일 이동평균선 (SMA)
- **표준편차**: 2.0 (K=2)
- **데이터**: 실시간 가격 데이터

### 매매 로직
1. **매수 조건**: 현재 가격 < 하단 밴드 (Lower Band)
2. **매도 조건**: 매수 포지션 보유 중, 가격이 중앙선(Middle Band) 도달
3. **거래 금액**: 사용자 설정 비율 사용

## 기술 스택
- **JavaScript/TypeScript**: 메인 개발 언어
- **Chrome Extension API**: 브라우저 확장 기능
- **DOM 조작**: 웹페이지 요소 인식 및 조작
- **Local Storage**: 데이터 저장
- **WebSocket**: 실시간 데이터 수신 (선택적)

## 주요 기능

### 매크로 시스템
- **버튼 위치 지정**: 사용자가 직접 클릭할 버튼 위치 지정
- **매크로 구성**: 매수/매도 시퀀스 구성
- **자동 실행**: 설정된 조건에 따라 매크로 자동 실행

### 기술적 분석
- **볼린저 밴드**: 실시간 계산 및 신호 생성
- **다양한 지표**: RSI, MACD, 이동평균선, 스토캐스틱
- **신호 통합**: 여러 지표의 신호를 종합하여 매매 결정

### 사용자 인터페이스
- **팝업 인터페이스**: 빠른 설정 및 상태 확인
- **옵션 페이지**: 상세 설정 및 매크로 구성
- **실시간 모니터링**: 현재 포지션 및 수익/손실 표시

### 데이터 관리
- **로컬 저장**: 설정 및 거래 내역 로컬 저장
- **백업/복원**: 설정 데이터 백업 및 복원
- **통계**: 거래 통계 및 성과 분석

## 설치 및 사용

### 1. 개발자 모드 활성화
1. Chrome 브라우저에서 `chrome://extensions/` 접속
2. "개발자 모드" 토글 활성화

### 2. 익스텐션 로드
1. "압축해제된 확장 프로그램을 로드합니다" 클릭
2. 프로젝트 폴더 선택

### 3. 거래소 설정
1. 익스텐션 팝업에서 거래소 선택
2. 매크로 구성 모드 활성화
3. 매수/매도 버튼 위치 지정

### 4. 전략 설정
1. 옵션 페이지에서 매매 전략 선택
2. 파라미터 설정
3. 알림 설정

## 보안 및 리스크 관리
- **로컬 실행**: 모든 데이터는 로컬에서만 처리
- **사용자 제어**: 매크로는 사용자가 직접 구성
- **리스크 경고**: 투자 위험에 대한 명확한 경고
- **백업 시스템**: 설정 및 데이터 백업 기능

## 확장 가능성
- **새로운 거래소 지원**: 매니페스트에 도메인 추가
- **추가 지표**: utils/indicators.js에 새 지표 추가
- **고급 매크로**: 복잡한 거래 시퀀스 지원
- **클라우드 동기화**: 설정 데이터 클라우드 백업
