# 크롬 익스텐션 기반 암호화폐 자동매매 봇

## 프로젝트 개요
텔레그램 신호를 받아 자동으로 거래를 실행하는 크롬 익스텐션 기반 암호화폐 자동매매 시스템입니다. 사용자가 직접 매크로를 녹화하여 거래소의 버튼을 자동으로 클릭하는 방식으로 작동합니다.

## 주요 특징
- **매크로 기반**: 사용자가 직접 버튼 위치 지정 및 매크로 구성
- **텔레그램 연동**: 텔레그램 봇을 통해 신호 수신 및 자동 거래 실행
- **다양한 거래소 지원**: Gate.io, Binance, Upbit, Coinbase 등
- **분할 진입/익절**: 3단계 분할 진입 및 분할 익절 지원
- **자동 스탑로스**: 진입 후 자동으로 스탑로스 설정 및 모니터링
- **사용자 친화적**: 직관적인 UI로 쉬운 설정
- **데이터 저장**: 로컬 스토리지 기반 설정 및 거래 내역 저장

## 기술 스택
- **JavaScript**: 메인 개발 언어
- **Chrome Extension API**: 브라우저 확장 기능
- **DOM 조작**: 웹페이지 요소 인식 및 조작
- **Chrome Storage API**: 데이터 저장
- **Telegram Bot API**: 텔레그램 신호 수신

## 주요 기능

### 매크로 시스템
- **버튼 위치 지정**: 사용자가 직접 클릭할 버튼 위치 지정
- **매크로 녹화**: Long, Short, Close 버튼 녹화
- **자동 실행**: 텔레그램 신호에 따라 매크로 자동 실행

### 텔레그램 연동
- **신호 수신**: 텔레그램 봇을 통해 거래 신호 수신
- **자동 거래**: 수신된 신호에 따라 자동으로 거래 실행
- **스크린샷 전송**: 거래 완료 시 스크린샷과 함께 결과 전송

### 분할 진입/익절
- **3단계 분할 진입**: 포지션을 3단계로 나누어 진입
- **분할 익절**: 3단계로 나누어 익절
- **트레일링 스탑**: 트레일링 스톱 기능

### 스탑로스 관리
- **자동 스탑로스**: 진입 후 자동으로 스탑로스 가격 설정
- **실시간 모니터링**: 현재가가 스탑로스 가격에 도달하면 자동 종료

### 거래 모드
- **One Way Mode**: 모든 비중이 진입되면 더 이상 진입하지 않음
- **Hedge Mode**: 진입 제한 없이 계속 시도

## 설치 및 사용

### 1. 개발자 모드 활성화
1. Chrome 브라우저에서 `chrome://extensions/` 접속
2. "개발자 모드" 토글 활성화

### 2. 익스텐션 로드
1. "압축해제된 확장 프로그램을 로드합니다" 클릭
2. 프로젝트 폴더 선택

### 3. 기본 설정
1. 거래소 선택 및 접속
2. 매크로 녹화 (Long, Short, Close)
3. 텔레그램 봇 설정 (Bot Ten, Chat ID, Symbol)
4. 거래 설정 (Leverage, Position, Stoploss, TP)

### 4. 자동매매 시작
1. "Auto Trading" 토글을 ON으로 설정
2. 텔레그램에서 거래 신호 전송

## 보안 및 리스크 관리
- **로컬 실행**: 모든 데이터는 로컬에서만 처리
- **사용자 제어**: 매크로는 사용자가 직접 구성
- **리스크 경고**: 투자 위험에 대한 명확한 경고
- **백업 시스템**: 설정 및 데이터 백업 기능
