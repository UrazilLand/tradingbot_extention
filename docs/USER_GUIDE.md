# 사용자 가이드

## 시작하기

### 첫 실행
1. **거래소 접속**: 지원하는 거래소 중 하나에 접속
2. **익스텐션 활성화**: 브라우저 툴바의 익스텐션 아이콘 클릭
3. **매크로 녹화**: Long, Short, Close 버튼 녹화
4. **텔레그램 설정**: Bot Token, Chat ID, Symbol 입력
5. **거래 시작**: "Auto Trading" 토글을 ON으로 설정

## 매크로 설정 가이드

### Long 매크로 녹화
1. **녹화 시작**: "Record" 토글을 ON으로 설정 후 "Long Record" 클릭
2. **요소 선택**: 거래소의 Long 버튼을 순서대로 클릭
3. **수량 입력**: 수량 입력 필드 클릭 (필요시)
4. **확인 버튼**: 주문 확인 버튼 클릭
5. **녹화 완료**: ESC 키를 눌러 녹화 완료

### Short 매크로 녹화
1. **녹화 시작**: "Record" 토글을 ON으로 설정 후 "Short Record" 클릭
2. **요소 선택**: 거래소의 Short 버튼을 순서대로 클릭
3. **수량 입력**: 수량 입력 필드 클릭 (필요시)
4. **확인 버튼**: 주문 확인 버튼 클릭
5. **녹화 완료**: ESC 키를 눌러 녹화 완료

### Close 매크로 녹화
1. **녹화 시작**: "Record" 토글을 ON으로 설정 후 "Record Close" 클릭
2. **요소 선택**: 거래소의 Close 버튼 클릭
3. **확인 버튼**: 주문 확인 버튼 클릭
4. **녹화 완료**: ESC 키를 눌러 녹화 완료

### 매크로 테스트
- 녹화 완료 후 "Manual" 모드에서 수동으로 테스트
- 거래소 페이지에서 Long/Short/Close 버튼이 정상 작동하는지 확인
- 오류 발생 시 매크로 재녹화

## 텔레그램 설정 가이드

### 텔레그램 봇 생성
1. 텔레그램에서 @BotFather 검색
2. `/newbot` 명령으로 새 봇 생성
3. 봇 이름과 사용자 이름 설정
4. Bot Token 복사

### Chat ID 확인
1. 텔레그램에서 @userinfobot 검색
2. 봇에게 메시지 전송
3. Chat ID 확인

### 익스텐션에 설정 입력
1. **Bot Token**: BotFather에서 받은 토큰 입력
2. **Chat ID**: userinfobot에서 확인한 ID 입력
3. **Symbol**: 거래할 심볼 입력 (예: BTC)
4. **Test 버튼**: 연결 테스트

## 거래 설정 가이드

### 기본 설정
- **Leverage**: 레버리지 배수 설정
- **Position**: 총 포지션 비율 (예: 30%)
- **Stoploss**: 손절 비율 (예: 2%)
- **TP Strategy**: 익절 전략 선택

### 분할 진입 설정
- **Position 1, 2, 3**: 각 단계별 진입 비율 입력
- 예: 33%, 33%, 34% (총 100%)
- 비율 합계가 100%가 되지 않으면 자동으로 재진입 시도

### 익절 전략 설정

#### Simple TP
- 단일 익절 비율 설정
- 예: 5% 설정 시 현재가가 진입가 대비 5% 상승하면 익절

#### Trailing TP
- **Trailing Distance**: 트레일링 거리 설정
- 예: 2% 설정 시 최고가 대비 2% 하락 시 익절

#### Split TP
- **TP 1, 2, 3**: 각 단계별 익절 비율 입력
- 예: 3%, 6%, 10%
- 각 단계별로 포지션의 일부씩 익절

### 거래 모드 설정
- **One Way Mode**: 모든 비중이 진입되면 더 이상 진입하지 않음
- **Hedge Mode**: 진입 제한 없이 계속 시도

## 자동매매 사용법

### 시작하기
1. 모든 설정 완료 후 "Auto Trading" 토글을 ON으로 설정
2. 텔레그램에서 거래 신호 전송
3. 신호 형식: `BTC LONG` 또는 `BTC SHORT`

### 신호 형식
- `{SYMBOL} LONG`: 롱 진입 신호
- `{SYMBOL} SHORT`: 숏 진입 신호
- 예: `BTC LONG`, `ETH SHORT`

### 자동 새로고침
1. "Auto Refresh" 설정에서 분 단위 입력
2. 0 입력 시 새로고침 비활성화
3. Auto Trading이 ON일 때만 작동

## 데이터 관리

### 데이터 추출
- **Extract Assets**: 현재 자산 추출
- **Extract Price**: 현재가 추출
- 버튼 클릭으로 자동으로 값이 입력됨

### 데이터 내보내기/가져오기
- **Export Data**: 모든 설정 및 매크로 내보내기
- **Import Data**: 백업 파일 가져오기

## 문제 해결

### 매크로가 작동하지 않을 때
1. 매크로 재녹화
2. 거래소 페이지 새로고침
3. 익스텐션 재로드

### 텔레그램이 연결되지 않을 때
1. Bot Token과 Chat ID 확인
2. Test 버튼으로 연결 테스트
3. 텔레그램 봇이 활성화되어 있는지 확인

### 거래가 실행되지 않을 때
1. 매크로가 모두 녹화되었는지 확인
2. Auto Trading이 ON인지 확인
3. 텔레그램 신호 형식 확인
4. 콘솔 로그 확인 (F12)
