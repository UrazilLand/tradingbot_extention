# 크롬 익스텐션 기반 암호화폐 자동매매 봇 사용자 가이드

## 시작하기

### 첫 실행
1. **거래소 접속**: 지원하는 거래소 중 하나에 접속
2. **익스텐션 활성화**: 브라우저 툴바의 익스텐션 아이콘 클릭
3. **매크로 설정**: 매수/매도 매크로 녹화
4. **전략 선택**: 원하는 매매 전략 선택
5. **거래 시작**: "거래 시작" 버튼 클릭

## 매크로 설정 가이드

### 매수 매크로 설정
1. **녹화 시작**: 팝업에서 "매수 매크로 녹화" 클릭
2. **요소 선택**: 매수 버튼을 순서대로 클릭
3. **수량 입력**: 수량 입력 필드 클릭 (필요시)
4. **확인 버튼**: 주문 확인 버튼 클릭
5. **녹화 완료**: ESC 키를 눌러 녹화 완료

### 매도 매크로 설정
1. **녹화 시작**: 팝업에서 "매도 매크로 녹화" 클릭
2. **요소 선택**: 매도 버튼을 순서대로 클릭
3. **수량 입력**: 수량 입력 필드 클릭 (필요시)
4. **확인 버튼**: 주문 확인 버튼 클릭
5. **녹화 완료**: ESC 키를 눌러 녹화 완료

### 매크로 테스트
- 녹화 완료 후 매크로가 정상 작동하는지 테스트
- 거래소 페이지에서 수동으로 매크로 실행
- 오류 발생 시 매크로 재녹화

## 매매 전략 가이드

### 볼린저 밴드 전략 (권장)
**적합한 시장**: 횡보장, 변동성이 큰 시장
**설정값**:
- 기간: 20 (기본값)
- 표준편차: 2.0 (기본값)
- 거래 비율: 10% (권장)

**매매 로직**:
- 매수: 가격이 하단 밴드 아래로 떨어질 때
- 매도: 가격이 중앙선에 도달할 때

### RSI 과매도 전략
**적합한 시장**: 강한 트렌드가 있는 시장
**설정값**:
- 기간: 14 (기본값)
- 과매도: 30 (기본값)
- 과매수: 70 (기본값)

**매매 로직**:
- 매수: RSI가 30 아래로 떨어질 때
- 매도: RSI가 70 위로 올라갈 때

### MACD 크로스오버 전략
**적합한 시장**: 트렌드가 명확한 시장
**설정값**:
- Fast: 12 (기본값)
- Slow: 26 (기본값)
- Signal: 9 (기본값)

**매매 로직**:
- 매수: MACD가 시그널선을 상향 돌파할 때
- 매도: MACD가 시그널선을 하향 돌파할 때

### 이동평균선 크로스 전략
**적합한 시장**: 장기 트렌드 시장
**설정값**:
- 단기: 10 (기본값)
- 장기: 20 (기본값)

**매매 로직**:
- 매수: 단기 MA가 장기 MA를 상향 돌파할 때
- 매도: 단기 MA가 장기 MA를 하향 돌파할 때

### 스토캐스틱 과매도 전략
**적합한 시장**: 변동성이 큰 시장
**설정값**:
- K%: 14 (기본값)
- D%: 3 (기본값)
- 과매도: 20 (기본값)
- 과매수: 80 (기본값)

**매매 로직**:
- 매수: K%가 20 아래로 떨어질 때
- 매도: K%가 80 위로 올라갈 때

## 고급 설정

### 거래 비율 조정
- **보수적**: 5-10% (안전한 거래)
- **적극적**: 10-20% (수익 추구)
- **공격적**: 20%+ (고위험 고수익)

### 체크 간격 설정
- **빠른 반응**: 5-10초 (단기 거래)
- **균형**: 30-60초 (일반적)
- **안정적**: 2-5분 (장기 거래)

### 알림 설정
- **모든 알림**: 모든 거래 신호 알림
- **중요 알림**: 매수/매도 신호만 알림
- **오류만**: 오류 발생 시만 알림

## 모니터링 및 관리

### 실시간 모니터링
1. **팝업 확인**: 익스텐션 팝업에서 상태 확인
2. **가격 추적**: 현재가 및 지표 값 확인
3. **포지션 관리**: 현재 포지션 상태 확인
4. **알림 확인**: 거래 신호 알림 확인

### 거래 내역 확인
1. **팝업에서**: "거래 내역" 버튼 클릭
2. **상세 정보**: 거래 시간, 가격, 수익/손실 확인
3. **통계 분석**: 승률, 평균 수익률 확인
4. **데이터 내보내기**: CSV 형태로 내보내기

### 성과 분석
- **총 거래 횟수**: 전체 거래 횟수
- **수익 거래**: 수익을 낸 거래 횟수
- **승률**: 수익 거래 비율
- **총 수익/손실**: 전체 수익/손실 금액
- **평균 수익/손실**: 거래당 평균 수익/손실

## 문제 해결

### 매크로 관련 문제

#### 매크로가 실행되지 않음
**원인**: 거래소 페이지 구조 변경
**해결방법**:
1. 매크로 재녹화
2. 거래소 페이지 새로고침
3. 다른 브라우저에서 테스트

#### 매크로 실행 중 오류
**원인**: 요소를 찾을 수 없음
**해결방법**:
1. 매크로 테스트 실행
2. 오류 발생 요소 확인
3. 해당 부분만 재녹화

### 거래 관련 문제

#### 거래 신호가 발생하지 않음
**원인**: 충분한 가격 데이터 없음
**해결방법**:
1. 거래소 페이지에서 충분한 시간 대기
2. 다른 전략 시도
3. 설정값 조정

#### 잘못된 거래 신호
**원인**: 전략 파라미터 부적절
**해결방법**:
1. 전략 파라미터 조정
2. 다른 전략 시도
3. 백테스팅으로 검증

### 성능 관련 문제

#### 브라우저가 느려짐
**원인**: 메모리 사용량 증가
**해결방법**:
1. 브라우저 재시작
2. 불필요한 탭 닫기
3. 다른 익스텐션 비활성화

#### CPU 사용량 증가
**원인**: 체크 간격이 너무 짧음
**해결방법**:
1. 체크 간격 늘리기 (최소 30초)
2. 불필요한 기능 비활성화
3. 시스템 리소스 확인

## FAQ

### Q: 어떤 거래소에서 사용할 수 있나요?
A: Gate.io, Binance, Upbit, Coinbase 등 주요 거래소에서 사용 가능합니다.

### Q: 매크로는 어떻게 작동하나요?
A: 사용자가 직접 버튼 위치를 지정하고, 익스텐션이 해당 버튼을 자동으로 클릭합니다.

### Q: 안전한가요?
A: 모든 데이터는 로컬에만 저장되며, 외부 서버로 전송되지 않습니다.

### Q: 수수료는 있나요?
A: 익스텐션 자체는 무료이며, 거래소 수수료만 발생합니다.

### Q: 모바일에서 사용할 수 있나요?
A: 현재는 데스크톱 Chrome 브라우저에서만 사용 가능합니다.

### Q: 백업은 어떻게 하나요?
A: 설정 및 데이터는 자동으로 로컬에 저장되며, 수동 백업 기능도 제공됩니다.
