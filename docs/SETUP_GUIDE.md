# 크롬 익스텐션 기반 암호화폐 자동매매 봇 설치 가이드

## 프로젝트 개요
사용자가 버튼 위치를 지정하고 매크로를 구성할 수 있는 크롬 익스텐션 기반 암호화폐 자동매매 시스템입니다. 볼린저 밴드 전략을 기반으로 하며, 다양한 거래소에서 작동합니다.

## 주요 특징
- **매크로 기반**: 사용자가 직접 버튼 위치 지정 및 매크로 구성
- **다양한 매매 전략**: 볼린저 밴드, RSI, MACD, 이동평균선, 스토캐스틱 등
- **다중 거래소 지원**: Gate.io, Binance, Upbit, Coinbase 등
- **실시간 알림**: 브라우저 알림 및 팝업 알림
- **사용자 친화적**: 직관적인 UI로 쉬운 설정
- **데이터 저장**: 로컬 스토리지 기반 설정 및 거래 내역 저장

## 설치 방법

### 1. 개발자 모드 활성화
1. Chrome 브라우저를 열고 주소창에 `chrome://extensions/` 입력
2. 우상단의 "개발자 모드" 토글을 활성화
3. "압축해제된 확장 프로그램을 로드합니다" 버튼 클릭

### 2. 프로젝트 폴더 선택
1. 프로젝트 폴더(`tradingbot_extension`) 선택
2. "폴더 선택" 클릭
3. 익스텐션이 설치되고 활성화됨

### 3. 익스텐션 확인
1. 브라우저 툴바에 익스텐션 아이콘 확인
2. 아이콘 클릭하여 팝업이 정상적으로 열리는지 확인

## 사용 방법

### 1. 거래소 접속
지원하는 거래소 중 하나에 접속:
- [Gate.io](https://www.gate.io)
- [Binance](https://www.binance.com)
- [Upbit](https://www.upbit.com)
- [Coinbase](https://www.coinbase.com)

### 2. 매크로 설정
1. 익스텐션 팝업에서 "매수 매크로 녹화" 또는 "매도 매크로 녹화" 클릭
2. 녹화 모드에서 클릭할 버튼들을 순서대로 선택
3. ESC 키를 눌러 녹화 완료

### 3. 전략 설정
1. 팝업에서 원하는 매매 전략 선택
2. 거래 비율 조정
3. "거래 시작" 버튼 클릭

### 4. 모니터링
- 팝업에서 실시간 상태 확인
- 거래 내역 및 통계 확인
- 알림을 통한 거래 신호 확인

## 지원하는 매매 전략

### 1. 볼린저 밴드 (기본)
- **매수 조건**: 현재 가격 < 하단 밴드
- **매도 조건**: 현재 가격 >= 중앙선
- **파라미터**: 기간(20), 표준편차(2.0)

### 2. RSI 과매도
- **매수 조건**: RSI < 30
- **매도 조건**: RSI > 70
- **파라미터**: 기간(14)

### 3. MACD 크로스오버
- **매수 조건**: MACD가 시그널선을 상향 돌파
- **매도 조건**: MACD가 시그널선을 하향 돌파
- **파라미터**: Fast(12), Slow(26), Signal(9)

### 4. 이동평균선 크로스
- **매수 조건**: 단기 MA가 장기 MA를 상향 돌파
- **매도 조건**: 단기 MA가 장기 MA를 하향 돌파
- **파라미터**: Fast(10), Slow(20)

### 5. 스토캐스틱 과매도
- **매수 조건**: K% < 20
- **매도 조건**: K% > 80
- **파라미터**: K(14), D(3)

## 설정 옵션

### 기본 설정
- **전략**: 사용할 매매 전략 선택
- **거래 비율**: 전체 자산 대비 거래 비율 (%)
- **체크 간격**: 가격 확인 주기 (초)

### 알림 설정
- **매수 신호**: 매수 신호 발생 시 알림
- **매도 신호**: 매도 신호 발생 시 알림
- **주문 체결**: 주문 체결 시 알림
- **포지션 청산**: 포지션 청산 시 알림
- **오류**: 오류 발생 시 알림

### 고급 설정
- **볼린저 밴드 기간**: 이동평균 기간 (5-100)
- **표준편차**: 표준편차 배수 (1.0-3.0)
- **RSI 기간**: RSI 계산 기간 (5-50)
- **MACD 파라미터**: Fast, Slow, Signal 기간

## 문제 해결

### 일반적인 문제

#### 1. 익스텐션이 작동하지 않음
- **해결방법**: 
  - 브라우저 새로고침
  - 익스텐션 재설치
  - 개발자 도구에서 오류 확인

#### 2. 매크로가 실행되지 않음
- **해결방법**:
  - 매크로 재녹화
  - 거래소 페이지 새로고침
  - 버튼 위치 변경 확인

#### 3. 가격 데이터를 가져올 수 없음
- **해결방법**:
  - 거래소 페이지 새로고침
  - 다른 거래소 시도
  - 익스텐션 재설치

#### 4. 알림이 표시되지 않음
- **해결방법**:
  - 브라우저 알림 권한 확인
  - 시스템 알림 설정 확인
  - 익스텐션 권한 확인

### 오류 코드

#### ERR_001: 매크로를 찾을 수 없음
- 매크로가 설정되지 않았거나 삭제됨
- 매크로 재녹화 필요

#### ERR_002: 요소를 찾을 수 없음
- 거래소 페이지 구조 변경
- 매크로 재녹화 필요

#### ERR_003: 가격 데이터 오류
- 거래소 연결 문제
- 페이지 새로고침 필요

#### ERR_004: 권한 오류
- 익스텐션 권한 확인
- 브라우저 재시작 필요
