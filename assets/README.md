# 아이콘 파일

이 폴더에 다음 크기의 아이콘 이미지를 추가하세요:

- `icon16.png` - 16x16 픽셀
- `icon32.png` - 32x32 픽셀  
- `icon48.png` - 48x48 픽셀
- `icon128.png` - 128x128 픽셀

## 아이콘 생성 방법

1. **온라인 도구 사용**:
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/
   - 128x128 크기의 원본 이미지를 업로드하면 모든 크기 자동 생성

2. **이미지 편집 도구 사용**:
   - Photoshop, GIMP, Paint.NET 등
   - 원본 이미지를 각 크기로 리사이즈

3. **무료 아이콘 다운로드**:
   - https://www.iconfinder.com/
   - https://www.flaticon.com/
   - 검색어: "crypto", "trading", "bot", "robot"

## 아이콘 디자인 팁

- **간단한 디자인**: 작은 크기에서도 알아볼 수 있도록
- **높은 대비**: 배경과 명확히 구분되도록
- **브랜드 컬러**: 프로젝트에 맞는 색상 사용
- **PNG 형식**: 투명 배경 지원

## 참고

아이콘 파일을 추가한 후:
1. Chrome Extension을 다시 로드
2. `chrome://extensions/`에서 새 아이콘이 표시되는지 확인
